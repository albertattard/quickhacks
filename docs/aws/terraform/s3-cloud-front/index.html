Terraform

ⓘ NoteDeployment may take several minute to create all required AWS resources. Don't be alarmed if a simple deployment of few files take several minutes to complete.

File: terraform.tf

The bucket quickhacks-terraform needs to be created beforehand. Refer to create S3 bucket for more information about that.

File: providers.tf

File: modules.tf

Used this module so that I can easily set the content type of the files. The files to be deployed are found in the folder ../web-app with respect to this file.

File: main.tf

File: output.tf

Access the default root object

Access the default root object, which is set to be the index.html page.

$ curl -v "https://$(AWS_PROFILE="quickhacks" terraform -chdir=terraform output -json domain_name | jq -r ".")"
