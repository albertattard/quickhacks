Create and Mount Vue Application

There are two approaches

  1. Create and mount the application from within the JavaScript
  2. Create the application from within the JavaScript and mount it from within the HTML

Both are valid approaches. The first one is more concise while the second one promotes reuse as the same script can be mounted to any element with needing to know its id.
